# 📌 API Module - 审计与文档更新 (2025-09-26)

## 概述
- 定义所有微服务间的Feign接口、DTO、VO等数据传输对象。
- 服务间调用的统一契约定义。

## 基本信息
- 模块名: api-module
- 类型: 接口定义库（非独立服务）

## 核心内容
- **Feign接口**: 各服务间调用的声明式HTTP客户端
- **DTO**: 数据传输对象
- **VO**: 视图对象
- **Request/Response**: 请求响应模型

## 设计原则
- 接口设计遵循RESTful规范
- DTO字段命名采用驼峰命名
- 版本控制通过包路径区分（v1、v2等）
- 统一异常响应格式
